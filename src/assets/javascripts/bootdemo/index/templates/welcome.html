Spring Boot App with AngularJS using Asset Pipeline, Spring Security and AuditAware

List of features this project have:

  • Single page app in which angular code is well managed based on modules
  • Spring Security - Authentication from database
  • AuditorAware based on Spring Security for tracking who last modified entity

For the demo, two user's have been created. Login with foloowing credentials:

  1. Admin User having ROLE_ADMIN. Since this is admin user, it has insert/update/delete rights. Credentials are:

    Username: admin@boot.com
    Password: admin

  2. Normal User having ROLE_USER. This user has viw only rights and does not have insert/update/delete rights. Credentials are:

    Username: user@boot.com
    Password: user
